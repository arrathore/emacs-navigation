# Emacs Navigation for Firefox

Adds Emacs-like navigation shortcuts to Firefox!  
Currently only supported for Mac.

Now available in the [Firefox Add-On Store](https://addons.mozilla.org/en-US/firefox/addon/emacs-navigation-for-firefox/)!

Shortcuts:
|Shortcut|Action|
|---|---|
|C-n|Move down a little|
|C-p|Move up a little|
|C-v|Scroll the page up almost one screen|
|M-v|Scroll the page down almost one screen|
|C-f|Move to the right a little|
|C-b|Move to the left a little|
